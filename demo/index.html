This is a particle engine / editor.

It's still very much in the works, you can report any bugs here:

https://github.com/gasolinewaltz/particleEngine

The dat.gui editor is interfacing with the engine and provides access to all of the editable variables.

  • stopped - the flag that controls whether the emitter is adding particles to the collection
  • cycleOnce - when set to true, the emitter will only cycle once
  • exportParticle - brings up some JSON that describes a particle. As of now, you still have to set emitter position / collection max / density programmatically. But you can still edit them with dat GUI.
  • reset - resets the cycle and clears the collection

The rest you'll just have to play around with :)
